# 🌟 About This Repository

This repository contains my complete **Generative AI learning journey**, where I explore modern AI technologies from the ground up.

It includes:

- 🤖 Large Language Models (LLMs)
- 💬 Prompt Engineering
- 🔗 LangChain
- 🌐 LangGraph
- 📚 Retrieval-Augmented Generation (RAG)
- 🧠 Embedding Models
- 🗄️ Vector Databases
- ⚡ Google Gemini API
- 🤖 AI Agents
- 📄 Document Q&A
- 🔥 Real-world GenAI Projects
- 🚀 FastAPI Deployment
